# 📚 Key Learnings From Building My Data Science Roadmap

## 1. Data Science Requires More Than Python

Initially, I thought Python was the main requirement.

After researching the field, I learned that Data Science also requires:

- Statistics
- Probability
- SQL
- Data Analysis
- Data Visualization
- Machine Learning
- Communication Skills

---

## 2. Statistics Is a Core Foundation

Machine learning algorithms rely heavily on statistical concepts.

Important areas include:

- Mean, Median, Mode
- Variance
- Standard Deviation
- Probability
- Hypothesis Testing

---

## 3. SQL Is Essential

Almost every Data Scientist works with databases.

Key SQL concepts:

- SELECT
- WHERE
- GROUP BY
- JOIN
- ORDER BY

---

## 4. Projects Matter More Than Certificates

Recruiters often evaluate:

- Portfolio projects
- GitHub repositories
- Problem-solving ability

More than certificates alone.

---

## 5. Machine Learning Comes After Data Analysis

I learned that jumping directly into Machine Learning is ineffective.

Recommended sequence:

1. Python
2. Statistics
3. SQL
4. Data Analysis
5. Visualization
6. Machine Learning

---

## 6. Networking Is Part of the Career

Building technical skills alone is not enough.

Professional visibility matters through:

- LinkedIn
- GitHub
- Kaggle
- Data Science communities

---

## 7. Consistency Beats Intensity

A structured learning schedule over 24 months is more sustainable than short bursts of studying.

---

## 8. Portfolio Development Should Start Early

Projects should be built alongside learning rather than waiting until all topics are completed.
